# oneup

CalVer-based version management for npm packages and Rust crates.

## Skills

| Skill | Description |
|-------|-------------|
| **oneup** | CalVer-based version management with oneup |

### Claude Code

```bash
# Add marketplace
/plugin marketplace add circlesac/oneup

# Install plugin
/plugin install oneup
```

### Pi

```bash
pi install npm:@circlesac/oneup
```
